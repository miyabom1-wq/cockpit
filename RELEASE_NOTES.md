# VANTAGE v47 — 日本株・信用需給

## 追加内容

- JPX「銘柄別信用取引週末残高」から、日本株の信用買残・信用売残・前週比・信用倍率を取得。
- 信用買残を20日平均出来高で割った「買残回転日数」を算出。
- 株価の5日変化と信用買残の前週変化を組み合わせ、`需給追い風 / 需給改善 / 中立 / 需給悪化 / 需給警戒` に分類。
- 日々公表、特別周知、信用取引規制を別々のフラグとして表示。
- 信用需給はA〜Eの元判定を変更せず、同一レーン内の確認順・注意点・ポジション上限にだけ補助利用。
- 候補、レーダー、ランキング、探索、ウォッチ、AI分析用コピーへ需給情報を追加。
- 管理タブに「信用需給」を追加し、登録銘柄の悪化・改善と市場全体の注意／規制銘柄を一覧化。
- VANTAGEからFRAMEへ、需給評価、信用倍率、買残、前週変化、買残回転日数、基準日、注意フラグを引き継ぎ。

## 判定上の扱い

信用需給は価格・市場差RS・出来高より下位の補助情報です。信用倍率が高いだけでA/B/Cを変更しません。

- 株価上昇＋買残減少：改善要因
- 株価下落＋買残増加：悪化要因
- 買残が平均出来高3〜5日分以上：戻り売り警戒
- 信用取引規制・特別周知：追加停止／サイズ縮小
- 日々公表：注意喚起。規制とは別
