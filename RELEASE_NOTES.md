# VANTAGE v41 Release Notes

## 登録ユニバースの安全な自動入れ替え

- 売買代金ランキング、探索、相対強度、出来高、候補レーン、ランキング継続日数を統合
- 日本株は現在の登録数を維持したまま弱い銘柄と有力候補を交換
- 米国株は先頭40のリード枠を維持し、退出対象は即削除せず保管枠へ降格
- 保有中、ウォッチ中、手動固定した銘柄は自動退出しない
- 重要テーマは最低2銘柄を維持し、ゼロまたは1銘柄のテーマを画面で警告
- ランキング履歴7日以上、候補とのスコア差18以上でのみ適用
- 週あたり日本・米国それぞれ最大2組まで
- 管理画面に「ユニバース」を追加し、提案・理由・テーマ構成・履歴を表示
- 自動、提案のみ、停止を選択可能

## UI v40相当の刷新を同梱

- VANTAGEはネイビー×ティール、FRAMEはチャコール×ブルーへ統一
- カード、余白、タブ、ボタン、PWAアイコンを刷新
- モーダル右上を明確な × に変更
- ISO形式の生時刻と内部確認件数を通常画面から除去
- テーマへ分析確度、JP/US母数、地域別確度を追加

## 分析上限

- 日本：最大160、コア80＋レーダー最大80
- 米国：登録上限80、先頭40を自動分析
- 自動入れ替えは日本の登録数を勝手に増やさない
- 米国リード枠は40のまま

## 自動更新

既存の市場更新時刻は変更しない。日本株の探索完了後、取引日の18:20頃にユニバース提案を更新する。安全自動モードでは同じ週に一度だけ適用する。

## 維持されるもの

- ウォッチ、シグナル、バックテスト、イベント
- VANTAGE → FRAME連携
- FRAMEの局面、打診、標準、追加、保有判定
- 既存KVデータ
