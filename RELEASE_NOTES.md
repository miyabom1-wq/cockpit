# VANTAGE v35 Release Notes

## 完全差し替え

- Worker正本を `worker/src/index.js` に一本化
- フロント正本を `public/index.html` に一本化
- 旧バンドル、重複エントリーポイント、`node_modules`、`.wrangler` を排除

## 分析

- ライブとバックテストの共通分析エンジン
- RSI14・ATR14をWilder RMAへ統一
- 配当調整をテクニカルOHLCへ混ぜない
- 株式分割の不連続だけを整合化
- 取引日・終値確定・OHLC・出来高・snapshot IDを監査
- データ無効時はA/B/C判定を停止

## 更新

- Stageを一時領域へ分割保存し、完成後に一括公開
- 日本確定終値の17:00再確認
- 米国確定終値の再確認を追加
- ランキング／探索35銘柄はベンチマークを共有し、外部通信37回に抑制

## ウォッチ・シグナル

- ウォッチ価格の多段フォールバック
- 個別再判定結果を全体Stageより優先可能
- 日本株・米国株を分離
- 検出当日の0.0%表示を廃止
- 確定終値のみで1・3・5営業日結果を更新

## バックテスト

- 5日線割れを翌営業日始値で執行
- トリガーとストップの同日順序不明を除外
- 新versionキーで再集計
- 完了後の不要なKV書き込みを停止
