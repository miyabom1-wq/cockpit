# VANTAGE v41 使い方

## 1. 市場

世界地合い、日経平均、S&P500、SOX、KOSPI/KOSDAQ、VIX、重要日程を確認する。リスクオフではA/B候補でも新規サイズを抑える。

## 2. テーマ

世界横断を基本に、発芽・修復・拡大・過熱・崩壊と1・3・5日変化を確認する。日米50:50を基準とし、片側3銘柄未満は中立方向へ補正する。

テーマ構成の「未登録」「少数」が出た場合は、分析母数が不足している。ユニバース管理がランキングと探索から補充候補を探す。

## 3. ユニバース

管理 → ユニバースを開く。

- 追加候補：ランキング、探索、RS、出来高、継続性が強い未登録銘柄
- 退出候補：ランキング圏外が続き、D/E判定やRS劣後が重なる銘柄
- 固定：銘柄編集で固定すると自動退出しない
- 保有・ウォッチ：自動的に保護される
- 米国退出：リード40から保管へ移すだけで即削除しない

### モード

- 週次・安全自動：提案を毎営業日更新し、条件を満たす場合に週1回、各市場最大2組を適用
- 提案のみ：候補を表示するが自動適用しない
- 停止：提案更新と自動適用を止める

初期設定は「週次・安全自動」。ランキング履歴が7日未満の場合は提案だけを表示し、適用しない。

## 4. FRAMEへ送る

テーマの先導銘柄または候補詳細から「FRAMEで判定」を押す。VANTAGEの文脈は引き継ぐが、FRAMEの局面・打診・標準・追加判定は独立している。

## 分析対象

- 日本：最大160。先頭80がコア、81〜160がレーダー
- 米国：登録上限80。先頭40をリードとして自動分析
- 自動入れ替えは現在の日本登録数を維持するため、無料運用の負荷を勝手に増やさない
- 市場全銘柄を常時分析する方式ではなく、ランキング・探索で広く検知し、登録ユニバースで詳細分析する
