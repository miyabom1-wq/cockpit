# VANTAGE v35 使い方

## 1. Stage

### サマリー

市場全体の向き、過熱度、重要日程を先に確認します。

```text
上向き      追い風。ただし個別の過熱は別判定
中立        個別選別
弱い        新規を抑え、A/Bでもサイズを落とす
```

### 候補ボード

```text
A  上昇レジーム内の強い継続候補
B  短期反転・再加速の初動候補
C  25日線近辺の押し目監視
D  条件未達またはデータ異常
E  急落、巨大上ヒゲ、過熱などの警戒
```

A/Bは買い指示ではありません。監査カードとチャートを確認する優先順位です。

### 監査カード

銘柄を開くと、次を確認できます。

```text
取引日
確定終値か場中暫定か
snapshot ID
データ有効／無効と理由
OHLC・出来高
SMA25/50/200
EMA65
RSI14 Wilder
ATR14 Wilder
市場RS5/20
各候補条件の合否
```

データ無効の銘柄はA/B/Cへ入りません。

### ランキング

売買代金上位の確認用です。
絶対順位だけでなく、新規ランクインと順位上昇を見ます。

### 探索

未登録の日本株から、売買代金、順位変化、出来高、市場RS、日足レジームを組み合わせて候補を出します。
勝手に登録せず、承認した銘柄だけレーダーへ入ります。

## 2. ウォッチ

日本株と米国株を別表示します。

```text
追跡中  通常監視
押し待ち  条件が整うまで待機
見送り  当面対象外
```

「価格・判定を更新」は、その銘柄だけYahoo日足を再取得し、最新のStage判定を保存します。
全体Stageより手動更新結果が新しければ、手動結果を優先表示します。

価格欄は現在取得できる場中値、または最新確定終値です。
データ日と確定状態も併記します。

---

## 3. シグナル検証

基準価格は、AまたはBを最初に検出した日の確定終値です。

```text
当日       翌営業日の終値待ち
1営業日後  基準終値との増減
3営業日後  基準終値との増減
5営業日後  基準終値との増減
```

当日は同じ価格を比較して0.0%と出さず、結果待ちと表示します。

```text
今日のA/B候補           現在も条件内
過去シグナル・5日観測中 A/B条件は終了したが結果追跡中
5日結果確定             5営業日分の結果が完成
```

「条件終了」は売却や失敗ではありません。
A/Bという検出期間が終わった意味です。

MFEは観測期間中の最大上昇率、MAEは最大下落率です。
日本株と米国株は混ぜずに集計します。

## 4. バックテスト

現在の登録銘柄を対象に、同じ分析エンジンで過去を再生します。

```text
A初回
B初回
C→A
C→B
```

主な前提:

```text
エントリー  翌営業日にシグナル日高値を突破
ギャップ    +3%超は見送り
ストップ    シグナル日安値
5日線割れ   終値で確認後、翌営業日始値で決済
同日順序不明 トリガーとストップ両方接触は除外
コスト      往復0.15%
```

勝率だけでなく、期待値、中央値、PF、件数を同時に見ます。
現在登録銘柄を使うため、生存者バイアスがあります。

## 5. 自動更新

### 日本株

```text
9:30   寄り後
10:20  前場
12:00  前場後・全体
14:20  後場
15:20  引け付近
16:10  確定終値
17:00  終値再確認
13:05 / 17:55 探索
```

### 米国株（夏時間）

```text
22:40 場中開始確認
5:20  確定終値
6:05  終値再確認
```

冬時間は1時間後です。

Cronは5分ごとに1作業単位を進めるため、全銘柄の完成には数十分かかります。
公開画面は途中世代へ切り替わらず、前回の完全snapshotを維持します。
